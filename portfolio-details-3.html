Data Analysis of Student's Exam Performance

  • Category: Data Analysis
  • Project Duration: Aug 2020 - Oct 2020

Participated in a data analysis challenge in order to compare the accuracy result between Random Forest, Logistic Regression and Decision Trees algorithm. In addition, I created a predictive model to predict student's academic performances based on various factours through utilizing linear regression and tensorflow keras. The tools that I used were:

  • Python
  • Pandas
  • Tensorflow
